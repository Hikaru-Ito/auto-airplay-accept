"iPhone" がAirPlayでこのMacに接続しようとしています
この接続を許可しますか？

毎回のクリックを、設計で消す。

バックグラウンドで小さく動き続けて、必要な瞬間にだけ仕事をする。

ゼロ・インタラクション

AirPlayダイアログが出た瞬間に自動で「受け入れる」をクリック。あなたは何もしなくていい。

日本語・英語に対応

「受け入れる」「Accept」のどちらの表示でも検出。OSの言語設定を切り替えても動きます。

完全バックグラウンド

Dockにもメニューバーにも出ません。リソース消費はごくわずか。普段の作業の邪魔をしません。

すべてオープンソース

中身は数十行のAppleScript。コードを読んで安心してから使えます。怪しいことは一切していません。

30秒でセットアップ。

DMGを開いて、Applicationsに入れて、起動する。それだけ。

01

ダウンロード

GitHub Releasesから最新の .dmg をダウンロードします。

02

Applicationsへドラッグ

DMGを開いて、アプリを Applications フォルダにドラッグします。

03

起動して権限を許可

アプリを起動するとアクセシビリティ権限が求められます。許可してください。

04

ログイン項目に追加（任意）

システム設定 → 一般 → ログイン項目 に追加すれば、Mac起動時に自動で動きます。
